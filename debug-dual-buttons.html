🐛 Common Issues

Issue 1: Looking at wrong component
❌ Field components (TEXT_INPUT, EMAIL_INPUT) don't have add buttons
✅ Only container components (PANEL, FIELDSET, GROUP) have add buttons
Issue 2: Not hovering
❌ Buttons are invisible until hover
✅ Move your mouse over the component to reveal buttons
Issue 3: CSS overlap
❌ Buttons might be overlapping or too close
✅ They should have a 4px gap between them

🛠️ Quick Fix Attempts
